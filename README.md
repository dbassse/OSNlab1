# 🐸 Введение в нейронные сети 🐸


Этот репозиторий содержит решения трёх практических заданий по построению и обучению полносвязных (Dense) нейронных сетей для задач классификации изображений. Все задания выполнены в среде Google Colab с использованием фреймворка TensorFlow/Keras.

## 📁 Структура репозитория

- `l1z1.ipynb` – ноутбук для задания Lite: перебор гиперпараметров (количество нейронов, функция активации, размер батча) на наборе геометрических фигур (треугольник, круг, квадрат). Проведено 18 экспериментов, составлена сравнительная таблица точности.
- `l1z2.ipynb` – ноутбук для задания Pro: бинарная классификация изображений крестиков и ноликов. Построена модель, достигнута точность выше 95%.
- `l1z3.ipynb` – ноутбук для задания Ultra Pro: распознавание рукописной цифры, нарисованной от руки. Модель обучается на MNIST, затем предсказывает цифру с загруженного пользователем фото.
- `mnist_weights.weights.h5` – сохранённые веса модели MNIST (чтобы не переобучать при каждом запуске).
- `цифра.png` – пример загруженной рукописной цифры.
- Аналогичное выполнение заданий на PyTorch

## 🛠 Требования

Для запуска ноутбуков рекомендуется использовать **Google Colab**. Все необходимые библиотеки предустановлены в Colab.
